mod.json

The mod's metadata. Without this file, the mod will be ignored.

Member Description
string name The name of the mod.
string desc The description.
string author The author(s).
string version The version.

Example

{
   "name": "Example Mod",
   "desc": "Adds absolutely nothing of value.",
   "author": "Lila",
   "version": "1.0"
}
