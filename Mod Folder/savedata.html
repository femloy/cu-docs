savedata.ini

The mod's save data.

Here's the most barebones save file:
[Mod]
enabled="1.000000"
The game uses the enabled entry to decide whether to activate the mod or not.
You can add your own save entries to this if your mod needs to save anything.

Example

ini_open(MOD_PATH + "/saveData.ini");

// Read
global.times_jumped = ini_read_real("Mod", "times_jumped", 0);

// Write
ini_write_real("Mod", "times_jumped", global.times_jumped);

ini_close();
This reads the times_jumped entry from the save file (returning 0 if it doesn't exist), and then saves the value.
