language

Mods can add or replace text entries to existing languages.
Adding new languages will not be part of the mod loader.

Language files are .txt files, and they look like this:
lang = "en"

# This is a comment
example_text = "Hello there."
This would be the english.txt file.
The lang entry is very important. It decides which language to add to. The .txt file name doesn't actually matter.

In Cheesed Up, there's just two languages:
lang Language
en English
spa Spanish (Spain)
If you have custom languages installed, and you want to support them, check their lang entry.

Usage

In GML code, you can get the entry like this: 
lang_get_value("example_text")
// OR
lang_get_value_newline("example_text")
If your text needs multiple lines, use the newline one.

This returns a string containing whatever you set in the example_text entry.

Example

// Draw GUI event
draw_text(0, 0, lang_get_value("example_text"));
This shows our "Hello there." entry at the top left of the screen.
