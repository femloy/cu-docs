init.gml

Runs when your mod is enabled.

Keep in mind that, this can either run when the game starts, or in the pause menu.
In both situations, the player would not exist. You need to handle these cases appropriately.

The purpose of this script is for preparing stuff like sprites, globals, save data...
You can also spawn a controller object that handles stuff more than once, or deals with the player.

Example

global.spr_icon = sprite_add(MOD_PATH + "/sprites/icon.png", 1, false, false, 0, 0);
This code loads a sprite from a file located at MOD_PATH/sprites/icon.png, and stores it in a global variable for future use.
